# Sudoku AI

This is a Sudoku AI project developed as part of my coursework for the AI course in my university.

## Description

The Sudoku AI project aims to solve Sudoku puzzles using artificial intelligence techniques. It utilizes various algorithms and strategies to find the solution to a given Sudoku puzzle.

## Features

- Sudoku puzzle solver using AI techniques
- Interactive user interface to input and display Sudoku puzzles
- Performance metrics to measure the efficiency of the solver
